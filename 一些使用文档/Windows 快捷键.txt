1、基本操作快捷键
Ctrl + Z：    撤销
Ctrl + Y：    重做
Ctrl + S：    保存
Ctrl + F：    查找
Ctrl + F3：    查找上一个
F3：    查找下一个
Ctrl + R：    替换

2、光标选中快捷键
Shift + ← / → (方向键)：    光标向左/向右选中一个字符
Ctrl + Shift + ← / → (方向键)：    光标向左/向右选中一个单词
Shift + Home：    光标从当前位置一直选中到行首
Shift + End：    光标从当前位置一直选中到行尾
 
3、光标跳转快捷键
Home：   光标跳转至行首
End：     光标跳转至行尾
PgUp：    光标跳转至首行
PgDn：    光标跳转至末行
Ctrl + ← / → (方向键)：    光标向左/向右跳转一个单词
Ctrl + Home：    光标跳转至首行首个字符前面(文件开头)
Ctrl + End：    光标跳转至末行最后一个字符后面(文件结尾)
